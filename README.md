## SKAMDLE 
Wordle-like guessing game, where you guess characters and episodes from the SKAM universe.


### Setup
If you want to run this project locally, follow these steps:

```bash
git clone https://github.com/remiermi/skam-wordle.git
cd skam-wordle
npm install
npm run dev
```
### Tech Stack

- React
- Tailwind CSS
- Vite
